# Lab 1: Establishing the Sovereign Workstation

## Objective
In this lab, you will prepare a fresh Ubuntu 24.04 environment for Agentic Development. Since BIG-CCC systems require precise toolchains, you will install the Go compiler from source and verify your environment before building the SAAYN-Agent.

- Installing the Go programming language (v1.22+)
- Configuring the Linux PATH for Go binaries
- Cloning the SAAYN-Agent source code
- Performing the initial "Bootstrap" build

## Procedure

1.  We'll begin by updating the local package index.
    `student@ubuntu:~$` `sudo apt update && sudo apt upgrade -y`

2.  Install the necessary build tools.
    `student@ubuntu:~$` `sudo apt install git make build-essential -y`

3.  Download the Go binary archive (v1.22.1).
    `student@ubuntu:~$` `wget https://go.dev/dl/go1.22.1.linux-amd64.tar.gz`

4.  Extract the archive to `/usr/local`.
    `student@ubuntu:~$` `sudo tar -C /usr/local -xzf go1.22.1.linux-amd64.tar.gz`

5.  Configure your shell profile to recognize Go.
    `student@ubuntu:~$` `echo 'export PATH=$PATH:/usr/local/go/bin:~/go/bin' >> ~/.bashrc`
    `student@ubuntu:~$` `source ~/.bashrc`

6.  Verify the Go installation.
    `student@ubuntu:~$` `go version`
    ```text
    go version go1.22.1 linux/amd64
    ```

7.  Clone the SAAYN-Agent repository.
    `student@ubuntu:~$` `git clone https://github.com/sfeeser/saayn-agent.git`
    `student@ubuntu:~$` `cd saayn-agent`

8.  Perform the initial build using the provided Makefile.
    `student@ubuntu:~/saayn-agent$` `make build`
    ```text
    🏗️  Building saayn...
    ✅ Build complete: ./saayn
    ```

## Conclusion
You have successfully transformed a raw Linux instance into a SAAYN-compatible workstation. By installing Go manually and building from source, you have ensured that your "Driver's Seat" is properly calibrated for the labs ahead.
